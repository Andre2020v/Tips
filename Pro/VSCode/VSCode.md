## Folding
|    Keyboard         |   Function                                                        |
|---------------------|-------------------------------------------------------------------|
|      zf             | 折叠命令, 可以使用v 来选取要折叠的行，也可以用鼠标选择                |
|      zf{count}j     | 折叠命令, zf5j 向下折叠5行                                         |
|      zf%            | 折叠命令，在括号出可以折叠括号内所有部分                             |
|      zM             | 折叠所有，配合命令，zM  zo，打开最外的一层，zO 打开所有折叠             |
|      zc             | 折叠命令，e.g. 在括号内任意位置run可以折叠括号                      |
